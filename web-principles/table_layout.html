Absolute vs. Relative Links

An absolute link is a hyperlink with a URL, which has all the information needed to find a site or file on the Internet. Absolute links can take you anywhere on the web, and always include the domain of the site.

A relative link differs because it is only accessible relative to the current page you're on. So, the file must be located somewhere on the current website you're on. It also doesn't include the site's domain and looks more like a file path.

Understanding the difference between relative and absolute links is important because both are heavily utilized when creating websites, and most sites need the capability to link from both within the site, and outside of the site.
